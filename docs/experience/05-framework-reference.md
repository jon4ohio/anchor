## Operational protocol (ongoing work)

From Entry §4:

```
Understand → Decide → Execute → Review → Improve
```

| Phase | Contract |
|-------|----------|
| Understand | Entry |
| Decide | ADR |
| Execute | Spec + Playbook |
| Review | Review |
| Improve | Handoff |

Adoption uses a different sequence (Assess → Classify → Orient → Continue). See [ADR-003](../decisions/ADR-003-product-narrative.md).
