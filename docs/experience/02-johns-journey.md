# John's Journey

John Ohio builds **tasklight**, a personal CLI, with AI over six months. He already has ADRs, architecture notes, and `CLAUDE.md`.

Anchor does not ask him to start over.

## What John did not do

- Delete existing ADRs
- Migrate every doc into a new folder structure
- Replace `CLAUDE.md` entirely — he narrowed it to operating instructions

He oriented what existed, separated responsibilities, and filled gaps only where coordination was missing.
