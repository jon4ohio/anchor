# Anchor

Anchor is an open-source experiment in durable project coordination. It captures context that should persist across sessions so people and AI can continue work with shared understanding instead of starting from scratch every session.

Anchor preserves durable project context. It does not prescribe project execution.

**New here?**

1. Read [Project Entry](docs/project/entry.md).
2. Follow [Start Project](ai/playbooks/start-project.md).
3. Experience guides are currently under validation.

## What Anchor is not

- An AI workflow or agent orchestration framework
- Implementation guidance for adopting projects
- A prescription for tooling, editors, or integrations
